How the Emotion-Based Movie Recommendation System Works

1. Fetching Popular Movies

We retrieve a list of popular movies using The Movie Database (TMDb) API. This gives us a wide range of movies that are currently popular with audiences.

2. Collecting User Reviews

For each movie, we gather user reviews and analyze the text for sentiment and emotional impact. These reviews provide insights into how viewers feel about each movie.

3. Sentiment and Emotion Analysis

We use two sentiment analysis tools, VADER and TextBlob, to analyze each review. Here’s what each tool provides:

VADER: Measures the overall sentiment of each review, determining if it’s positive, neutral, or negative.

TextBlob: Detects polarity (positivity or negativity) and subjectivity, which reflects how strongly an opinion is expressed.

4. Creating Emotional Profiles

Based on the analysis, each movie is given an emotional profile with scores that represent categories like "uplifting," "intense," "dark," "thought-provoking," "romantic," and "humorous." These profiles help us understand the dominant emotions associated with each movie.

5. Matching Your Preferences

When you select an emotional preference, the system recommends movies with matching profiles. For example, if you choose "uplifting," the system will prioritize movies with high positive sentiment scores.

6. Continuous Improvement

We continuously refine the emotional analysis based on feedback and improve the accuracy of the recommendations. This way, the system becomes better at predicting the emotional impact of each movie over time.
